MCZ Maestro - Control your MCZ pellet stove from Homey

Control your MCZ Maestro pellet stove directly from Homey.

FEATURES:
- Turn stove on/off
- Set target temperature (7-35°C)
- Control thermostat modes (Manual, Auto, Dynamic, Turbo)
- Adjust fan speeds (3 independent fans)
- Monitor temperatures (ambient, exhaust, water)
- Enable/disable Eco mode
- Set manual power level (1-5)
- Alarm monitoring

REQUIREMENTS:
- MCZ pellet stove with Maestro+ technology
- Stove must be connected to MCZ cloud service
- MCZ Maestro+ account (create via mobile app)

SETUP:
1. Install this app on your Homey
2. Go to Devices → Add Device → MCZ Maestro
3. Enter your MCZ Maestro+ credentials (email and password)
4. Select your stove from the list
5. Done! Your stove is now connected

The app polls the MCZ cloud every 30 seconds to update status and accepts commands to control your stove.
